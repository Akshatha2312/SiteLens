## Project Title
SiteLens

## Project Description
SiteLens is a RAG-Powered Website Chatbot that crawls websites recursively, processes webpage content, generates embeddings, indexes the data with FAISS, and answers user questions using Google Gemini via Retrieval-Augmented Generation.

## Problem Statement
Many websites contain useful information, but searching through pages manually is slow and inefficient. SiteLens solves this by creating a searchable knowledge layer over a website and answering questions based only on the crawled content.

## Features
- Recursive website crawling using BeautifulSoup
- Webpage text extraction and processing
- Embedding generation for content vectors
- FAISS-based similarity indexing
- Retrieval-Augmented Generation (RAG) with Google Gemini
- Answering questions from crawled website content only
- FastAPI backend with React + Vite frontend

## Tech Stack
- React
- Vite
- FastAPI
- Python
- BeautifulSoup
- FAISS
- Google Gemini API

## System Architecture / Workflow
1. Crawl website content recursively.
2. Process and clean HTML text.
3. Generate embeddings for text chunks.
4. Index embeddings with FAISS.
5. Retrieve relevant content and answer questions using Gemini.

## Project Structure
- `backend/` — Python backend and RAG pipeline
  - `backend/app/` — API and RAG implementation
  - `backend/requirements.txt` — Python dependencies
  - `backend/.env` — environment variables for Gemini API and model
- `frontend/` — React + Vite user interface
  - `frontend/src/` — React components and app logic
  - `frontend/package.json` — frontend dependencies
- `assets/` — project assets and static files
- `debug_crawl.py` — utility script for local crawling/debugging

## Prerequisites
- Python 3.11+ installed
- Node.js 18+ installed
- Access to a Google Gemini API key
- `backend/.env` configured with `GEMINI_API_KEY` and `MODEL_NAME`

## Installation
1. Clone the repository.
2. Install backend dependencies:
   ```bash
   cd backend
   .\venv\Scripts\Activate.ps1
   python -m pip install -r requirements.txt
   ```
3. Install frontend dependencies:
   ```bash
   cd ..\frontend
   npm install
   ```

## How to Run the Backend
1. Activate the backend virtual environment:
   ```bash
   cd backend
   .\venv\Scripts\Activate.ps1
   ```
2. Start the FastAPI backend:
   ```bash
   uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
   ```
3. Confirm the backend is running at `http://127.0.0.1:8000`.

## How to Run the Frontend
1. Open the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies if not already installed:
   ```bash
   npm install
   ```
3. Start the React frontend:
   ```bash
   npm run dev -- --host 127.0.0.1 --port 4173
   ```
4. Open the application in your browser at `http://127.0.0.1:4173`.

## Usage Instructions
1. Start the backend and frontend services.
2. Open the frontend in a browser.
3. Enter a website URL to crawl.
4. Use the chatbot interface to ask questions about the crawled website.
5. The backend retrieves relevant content and returns answers generated by Gemini.

## Complete Workflow (Crawl → Process → Embed → Index → Ask)
1. Crawl
   - Traverse the target website recursively and gather page content.
2. Process
   - Clean and normalize HTML text from crawled pages.
3. Embed
   - Generate vector embeddings for content chunks.
4. Index
   - Store embeddings in a FAISS index for fast similarity search.
5. Ask
   - Query the FAISS index, retrieve relevant chunks, and generate answers via Gemini.
